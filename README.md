# Advent of Code 2022 in Rust

### Getting Started

Because the implementations use nightly-only experimental features, first set the channel to ``nightly``:
```
cargo default nightly
```

To run solutions, do
```
cargo run -- <day> <part>
```

For example, to run the solution from **Day 2 Part 1**, do:
```
cargo run -- 2 1
```
